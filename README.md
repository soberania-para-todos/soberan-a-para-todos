# Soberanía para Todos 🌍🔒

**Soberanía para Todos** es una comunidad abierta que construye herramientas prácticas para que cualquier persona o comunidad pueda depender menos de sistemas externos y más de sus propios recursos.

No es nacionalismo. Es **autonomía funcional**.

---

## 🧭 ¿Qué significa "soberanía para todos"?

Que una persona, una comunidad o un país pueda:

- ⚡ **Procesar su propia energía** (paneles solares, cooperativas)
- 💰 **Controlar su dinero** (Bitcoin, bots de trading, sin bancos)
- 🔐 **Proteger sus datos** (GrapheneOS, servidor propio)
- 🌽 **Producir sus alimentos** (huertos urbanos, compras comunitarias)
- 🛠️ **Usar tecnología libre** (software de código abierto, automatización)

No se trata de aislarse, sino de **no depender de quien no te debe nada**.
