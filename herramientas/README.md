# Herramientas para la Soberanía 🛠️

Esta carpeta contiene **scripts, guías y recursos prácticos** para que cualquier persona pueda avanzar hacia la soberanía en diferentes áreas: finanzas, datos, energía, alimentos y tecnología.

Cada herramienta está diseñada para ser:
- **Simple** (que funcione con pasos claros)
- **Accesible** (sin necesidad de conocimientos técnicos avanzados)
- **Autónoma** (que puedas ejecutarla sin depender de servicios externos)
